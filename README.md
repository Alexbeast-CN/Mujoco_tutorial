# Mujoco 杂记

## 前言

最近的新项目用的到 Mujoco，所以打算从工业界的角度记录一下我在学习和使用 Mujoco 过程中的一些经验和心得。进而创建了专栏 [Mujoco 杂记](https://www.zhihu.com/column/c_1808928206302093312) 。在使用 Mujoco 仿真的过程中，为了方便项目的开发，我编写了一些小工具和做了脱敏的功能示例程序，方便 Mujoco 的初学者参考。专栏的所有代码，全部位于：[Alexbeast-CN/Mujoco_tutorial](https://github.com/Alexbeast-CN/Mujoco_tutorial) 欢迎各位大佬为代码提交 issue 和 PR。
